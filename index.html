Welcome

Castor, a small solid Web-Framework written in PHP.

The build in classes couldt give a high scalable background for your own MVC or whatever concept. A xml-configuration couldt describe a complete Homepage and works powerfull with XSLT-Templates. Serveral additions to add constants and individual modules, prepares a data-model for a page.

Use case

As the productive base, a class named as "xsltDocument" loads Controller Classes that extend Application Objects. Data prepared by modules and "document elements", feeds Controller Classes with elements given for a Page. The data as array, DomDocument object or plaintext, couldt served and adopt in serveral data formats like xml and json. A little copy paste of the action configuration, transforms the hole base code from a dynamic page to an asynchronous service.
